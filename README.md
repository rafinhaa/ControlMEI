# ControlMEI

Software de gerencia para Micro Empreendedor Individual (MEI)
- Cadastro de empresa
- Cadastro de receita (Revenda de Mercadorias, Venda de produtos e Prestação de Serviços)
- Consulta a receitadas cadastradas
- Emissão de relatórios
